# Fonts & credits

StoryType bundles only **open-source fonts** licensed under the **SIL Open Font
License 1.1 (OFL)** or an equivalent free license. Each font's full license text
lives beside this file (e.g. `OFL-Vazirmatn.txt`).

> ⚠️ **Never add commercial Persian fonts** (IRANYekan / IRANYekanX, IRANSans,
> Yekan Bakh, Dana, Kalameh, Morvarich, etc.) to `/public/fonts`. Sites like
> cdnfonts.com, font.download and cufonfonts host pirated copies — their hosting
> is **not** a license. Only bundle fonts whose OFL/free license you can point to.
